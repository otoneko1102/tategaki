# tategaki
横書きの文章を擬似的に縦書きに変換します。
### 変換前

```
いろはにほへと
ちりぬるを
わかよたれそ
つねならむ
うゐのおくやま
けふこえて
あさきゆめみし
ゑひもせす
```
### 変換後
```
ゑあけうつわちい
ひさふゐねかりろ
もきこのなよぬは
せゆえおらたるに
すめてくむれをほ
　み　や　そ　へ
　し　ま　　　と
```
